# oasistools

A lightweight program manager for Windows.  Install, update, and manage oasis tools from one place. 

## Download

Get the latest version from the [Releases](https://github.com/oasisdfg/oasistools/releases) page.

| File | Description |
|------|-------------|
| Setup.exe | Installer with Start Menu and Desktop shortcuts |
| Portable.zip | Extract anywhere and run |

## Available Programs

| Program | Description |
|---------|-------------|
| Login Manager | Securely store and manage your login credentials |
| Hardware Monitor | Real-time CPU, GPU, RAM, network, and storage monitoring |

## Requirements

- Windows 10/11
- .NET 8.0 Runtime
